# KODLUYORUZ REACT NATIVE BOOTCAMP | FIRST PROJECT

## 📋 Requirements

- Node.js

## 🛠️ Create New a Project

Install Expo CLI:
`npm install -g expo-cli`

Create a new project:
`expo init first-project`

Open the file:
`cd first-project`

And just run it:
`expo start`

[For More Information](https://reactnative.dev/docs/environment-setup)
